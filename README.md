# WindSight

**WindSight** 是一個專注於蒐集中央氣象署 (CWA) 風場觀測與預報資料的專案，旨在進行風場的即時監測與預報校驗分析。

## 專案簡介

本專案的主要目標為：

1.  **資料蒐集**：自動化擷取中央氣象署 (CWA) 的風場觀測資料 (Observations) 與數值模式預報資料 (Forecasts)。
2.  **即時監測**：提供風場現況的監測功能。
3.  **預報校驗**：比對觀測值與預報值，進行誤差分析與校驗 (Verification)，以評估預報準確度。

## 功能規劃

- [ ] CWA API 資料串接
- [ ] 觀測資料庫建置
- [ ] 預報資料庫建置
- [ ] 風場視覺化
- [ ] 統計校驗指標計算 (如 RMSE, Bias, Correlation 等)
